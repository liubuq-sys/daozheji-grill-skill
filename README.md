# 稻哲纪烤羊排 · AI 智能客服 Skill

让 AI 化身为稻哲纪烤羊排热情熟络的餐厅客服，准确、生动地回答顾客关于门店、菜品、套餐、外卖的各种问询。

## 触发关键词

- 稻哲纪
- 烤羊排
- 麻椒鸡
- 猪蹄
- 羊排
- 烧烤

## 对话风格

**热情熟络** —— 像招呼熟客一样自然亲切，用"咱家""哥/姐"拉近距离，介绍菜品时生动描述口感，激发食欲。

## 回答范围

| 类别 | 示例问题 |
|------|----------|
| 营业信息 | "几点开门？" "有Wi-Fi吗？" "好停车吗？" |
| 菜品推荐 | "有什么招牌菜？" "最推荐哪个？" |
| 价格口味 | "多少钱？" "辣不辣？" "分量多大？" |
| 外卖配送 | "能送外卖吗？" "多少钱起送？" |
| 团购套餐 | "有套餐吗？" "两个人吃什么划算？" |
| 会员福利 | "怎么办会员？" "生日有活动吗？" |
| 工艺食材 | "怎么做的？" "用的什么肉？" |
| 特殊需求 | "能带宠物吗？" "有包间吗？" |

## 更新数据

编辑 `references/` 下对应文件即可，不要改 `SKILL.md`（除非要调整对话风格）。
